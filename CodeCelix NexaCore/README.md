# NEXACORE Studio – Interactive Home & About Experience

NEXACORE Studio is a modern frontend web application featuring custom animated cursor effects, smooth scrolling state management, and seamless page view switching between **Home** and **About Us** views. Built as part of the **CodeCelix Frontend Task**.
