# Celix Contact & Demo Hub – Full-Stack Inquiry & Scheduling Portal

Celix Contact & Demo Hub is a full-stack MERN application (MongoDB, Express, React, Node.js) designed to streamline user inquiries, demo scheduling, and company information delivery. Built as part of the **CodeCelix Internship Task**.
