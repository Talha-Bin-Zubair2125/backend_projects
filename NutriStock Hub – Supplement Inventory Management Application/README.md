# NutriStock Hub – Supplement Inventory Management Application

NutriStock Hub is a full-stack MERN application (MongoDB, Express, React, Node.js) featuring client-side multi-page routing with **React Router DOM**. It provides an end-to-end portal to browse, search, and register fitness supplements with real-time stock and expiration tracking.
