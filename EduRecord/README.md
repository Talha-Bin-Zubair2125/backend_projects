# EduRecord – Student Registration & Management System

EduRecord is a full-stack web application built using the **MERN** stack (MongoDB, Express, React, Node.js). It provides a clean, responsive interface for managing student information with complete CRUD (Create, Read, Update, Delete) functionality.

---

## Features

- 📝 **Register Students**: Add student profiles including Name, Father's Name, Course, and Semester.
- 📋 **View Records**: Fetch and display registered students dynamically in a structured table.
- ✏️ **Inline Updating**: Edit and update student records seamlessly using database IDs.
- 🗑️ **Delete Confirmation**: Remove student records with built-in confirmation prompts to prevent accidental deletion.
- ⚡ **RESTful API Architecture**: Efficient server-side operations utilizing Express.js routes and Mongoose ORM.

---

## Tech Stack

### Frontend
- **React.js** (Functional Components, Hooks, State Management)
- **Axios** (HTTP Client for API requests)
- **CSS3** (Custom Styling & Responsive Layout)

### Backend
- **Node.js** & **Express.js** (Server & RESTful API endpoints)
- **MongoDB** & **Mongoose** (Database & Schema Modeling)
- **CORS** (Cross-Origin Resource Sharing middleware)

## Getting Started

### Prerequisites

Ensure you have the following installed on your local machine:
- [Node.js](https://nodejs.org/) (v16.x or higher)
- [MongoDB Community Server](https://www.mongodb.com/try/download/community) running locally on port `27017`
