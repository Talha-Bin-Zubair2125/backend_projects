# CodeCelix Portal – Frontend Internship Task

This repository contains the frontend implementation completed as part of the **CodeCelix Internship Task**. The project features responsive frontend pages—including **Pricing & Plans**, **Partnerships**, and **How It Works**—built using **React**, **Vite**, and **Tailwind CSS**.
