# 🚀 Node.js & Express Backend Projects Collection

Welcome to the **`backend_projects`** repository! This repo serves as a central collection of Node.js, Express.js, and MERN stack applications, demonstrating backend concepts including RESTful APIs, authentication (JWT & Cookies), role-based access control (RBAC), database integration with MongoDB/Mongoose, middleware design, and proxy integrations.

---

## 🛠️ Main Tech Stack

- **Runtime Environment:** Node.js
- **Backend Framework:** Express.js
- **Database / ODM:** MongoDB, Mongoose
- **Authentication & Security:** JSON Web Tokens (JWT), HttpOnly Cookies, Bcrypt, CORS
- **Frontend / Integration:** React.js, Vite, Axios, React Router DOM

---

## 📁 Repository Directory & Projects Index

Below is an overview of the key backend projects and applications contained within this repository:

### 🔐 Authentication & Role-Based Systems
- **`PublishPulse – Role-Based Editorial`** – Full-stack publishing application with role-based workflows for Readers, Authors, and Admins.
- **`TaskPulse – Role-Based Task Management`** – Task management platform with admin assignment panels and user tracking.
- **`SecureAuth Express`** – Secure authentication service implementing signed `HttpOnly` cookie-based JWT sessions.
- **`User Registry App`** – User onboarding, authentication, and directory management API.

### 🎵 Real-Time APIs & Integrations
- **`TuneWave`** – Web-based music search and preview application using Express as a proxy for the Deezer API.
- **`SkyFetch Express`** – Express API integration handling external data fetching and weather/location queries.

### 📦 Management Systems & Micro-APIs
- **`NutriStock Hub` / `NutriStock`** – Supplement inventory and stock management backend with CRUD capabilities.
- **`ContactVault Express`** – Contact storage and management API.
- **`EduRecord`** – Academic records and student transcript management service.
- **`ticket_support_system`** – Helpdesk and support ticket routing backend.
- **`stg_intern_tracker`** – Internship workflow, task, and performance tracking system.

### ✍️ Content & Utility Tools
- **`BlogPress Express` / `BlogSphere API`** – RESTful blogging platforms with article creation, tagging, and draft features.
- **`MarkParse Express`** – Markdown parsing and rendering backend utilities.
- **`TextForge Express`** – String processing and text formatting microservices.
- **`CalcAPI Studio`** – Specialized RESTful calculation and data processing API service.
- **`ChronosSync Express`** – Time-series and scheduling sync helper API.
- **`code_review_and_snippet_manager`** – Developer code snippet vault and peer review tool.
